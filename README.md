# Hat Size Calculator

A browser-based tool for calculating hat dimensions and generating flat cutting patterns.

## Features

- Calculate bottom diameter from top diameter, height, and side angle
- Supports tapered, straight, and flared crown shapes
- Brim width and outer brim diameter calculation
- Scaled cross-section diagram
- Cutting pattern generator for crown side, top circle, and brim ring
- Seam allowance, alignment notches, and side seam overlap options
- Download patterns as SVG
- Supports cm, inches, and mm

## Usage

Open `index.html` in any modern browser — no build step or server required.
